# LeetCode Hot100 错题本

记录我刷 Hot100 时做错、想复杂、没想到关键思路的题。

这是一个用 GitHub Pages 发布的 Python 刷题笔记网站。`index.md` 是专题首页，具体笔记放在 `docs/` 目录下，按数组与字符串、栈、动态规划、二分查找、链表、贪心和矩阵等专题分别维护。

## 维护方式

- 在 Typora 中打开 `docs/` 目录下对应专题的 Markdown 文件继续记录。
- 新增题目时，放到最匹配的专题；没有合适分类时再创建新文件。
- 更新完成后运行 `sync-notes.bat`，即可提交并同步到 GitHub Pages。
