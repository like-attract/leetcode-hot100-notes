下面讲一下 **Python 中链表的基本操作**。Python 标准库没有像 C 语言那样的原生链表结构，但我们可以用 **类 + 节点对象** 自己实现。

---

## 1. 什么是链表？

链表是一种线性数据结构，由多个 **节点 ListNode** 组成。

每个节点通常包含两部分：

```python
data   # 存储数据
next   # 指向下一个节点
```

例如：

```text
head -> 10 -> 20 -> 30 -> None
```

其中：

- `head` 指向第一个节点
- 最后一个节点的 `next` 指向 `None`

---

## 2. 定义链表节点

```python
class ListNode:
    def __init__(self, data):
        self.data = data
        self.next = None
```

---

## 3. 定义链表类

```python
class LinkedList:
    def __init__(self):
        self.head = None
```

---

## 4. 在链表末尾添加节点

```python
def append(self, data):
    new_node = ListNode(data)

    if self.head is None:
        self.head = new_node
        return

    current = self.head
    while current.next is not None:
        current = current.next

    current.next = new_node
```

示例：

```python
ll = LinkedList()
ll.append(10)
ll.append(20)
ll.append(30)
```

结果：

```text
10 -> 20 -> 30 -> None
```

---

## 5. 在链表头部添加节点

```python
def prepend(self, data):
    new_node = ListNode(data)
    new_node.next = self.head
    self.head = new_node
```

示例：

```python
ll.prepend(5)
```

原来：

```text
10 -> 20 -> 30
```

之后：

```text
5 -> 10 -> 20 -> 30
```

## 6. 遍历链表

```python
def display(self):
    current = self.head
    result = []

    while current is not None:
        result.append(str(current.data))
        current = current.next

    print(" -> ".join(result))
```

示例：

```python
ll.display()
```

输出：

```text
10 -> 20 -> 30
```

---

## 7. 查找节点

```python
def search(self, data):
    current = self.head

    while current is not None:
        if current.data == data:
            return True
        current = current.next

    return False
```

示例：

```python
print(ll.search(20))  # True
print(ll.search(99))  # False
```

## 8. 删除指定值的节点

```python
def delete(self, data):
    if self.head is None:
        return

    if self.head.data == data:
        self.head = self.head.next
        return

    current = self.head

    while current.next is not None:
        if current.next.data == data:
            current.next = current.next.next
            return
        current = current.next
```

示例：

```python
ll.delete(20)
```

原来：

```text
10 -> 20 -> 30
```

之后：

```text
10 -> 30
```

## 9. 在指定位置插入节点

```python
def insert(self, index, data):
    new_node = ListNode(data)

    if index == 0:
        new_node.next = self.head
        self.head = new_node
        return

    current = self.head
    position = 0

    while current is not None and position < index - 1:
        current = current.next
        position += 1

    if current is None:
        print("Index out of range")
        return

    new_node.next = current.next
    current.next = new_node
```

示例：

```python
ll.insert(1, 15)
```

原来：

```text
10 -> 20 -> 30
```

之后：

```text
10 -> 15 -> 20 -> 30
```

## 10. 反转链表

```python
def reverse(self):
    prev = None
    current = self.head

    while current is not None:
        next_node = current.next
        current.next = prev
        prev = current
        current = next_node

    self.head = prev
```

示例：

```python
ll.reverse()
```

原来：

```text
10 -> 20 -> 30
```

之后：

```text
30 -> 20 -> 10
```

## 11. 完整示例

```python
class ListNode:
    def __init__(self, data):
        self.data = data
        self.next = None


class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = ListNode(data)

        if self.head is None:
            self.head = new_node
            return

        current = self.head
        while current.next is not None:
            current = current.next

        current.next = new_node

    def prepend(self, data):
        new_node = ListNode(data)
        new_node.next = self.head
        self.head = new_node

    def delete(self, data):
        if self.head is None:
            return

        if self.head.data == data:
            self.head = self.head.next
            return

        current = self.head

        while current.next is not None:
            if current.next.data == data:
                current.next = current.next.next
                return
            current = current.next

    def display(self):
        current = self.head
        result = []

        while current is not None:
            result.append(str(current.data))
            current = current.next

        print(" -> ".join(result))


ll = LinkedList()

ll.append(10)
ll.append(20)
ll.append(30)

ll.display()       # 10 -> 20 -> 30

ll.prepend(5)
ll.display()       # 5 -> 10 -> 20 -> 30

ll.delete(20)
ll.display()       # 5 -> 10 -> 30
```

## 12. 常见操作时间复杂度

| 操作     | 时间复杂度 |
| -------- | ---------- |
| 头部插入 | O(1)       |
| 尾部插入 | O(n)       |
| 查找元素 | O(n)       |
| 删除元素 | O(n)       |
| 遍历链表 | O(n)       |
| 反转链表 | O(n)       |

## 13. 链表和 Python 列表的区别

| 特性     | Python 列表  | 链表         |
| -------- | ------------ | ------------ |
| 内存     | 连续存储     | 分散存储     |
| 随机访问 | 快，O(1)     | 慢，O(n)     |
| 头部插入 | 慢，O(n)     | 快，O(1)     |
| 中间插入 | 需要移动元素 | 改指针即可   |
| 适合场景 | 普通数据列表 | 频繁插入删除 |
